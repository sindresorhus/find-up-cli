# find-up-cli

> Find a file by walking up parent directories

## Install

```
$ npm install --global find-up-cli
```

## Usage

```
$ find-up --help

  Usage
    $ find-up <filename>

  Options
    --cwd=<directory>  Working directory
```

## Example

```
/
└── Users
    └── sindresorhus
        ├── unicorn.png
        └── foo
            └── bar
                ├── baz
                └── faz
```

```
$ echo $PWD
/Users/sindresorhus/foo/bar/faz
$ find-up unicorn.png
/Users/sindresorhus/unicorn.png
```
